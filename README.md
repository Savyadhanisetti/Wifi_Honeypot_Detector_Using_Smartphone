# Wifi_Honeypot_Detector_Using_Smartphone

A smart cybersecurity project that detects suspicious and fake WiFi access points (honeypots) using a smartphone. The system scans nearby wireless networks and analyzes parameters such as SSID duplication, signal strength anomalies, MAC address behavior, encryption type, and network inconsistencies to identify potentially malicious hotspots created by attackers.

The project is developed using Python, Flask, and networking tools to provide real-time WiFi threat detection through a simple web-based interface. It helps users avoid connecting to fake public WiFi networks that may be used for data theft, phishing, or man-in-the-middle attacks.

**Features**
Scan nearby WiFi networks
Detect duplicate/fake SSIDs
Identify suspicious open networks
Analyze signal strength and security type
Real-time detection results
User-friendly web interface
Smartphone compatible access
**Technologies Used**
Python
Flask
HTML/CSS/JavaScript
Networking Commands (netsh)
Cybersecurity Concepts
WiFi Network Analysis
**Objective**

The main objective of this project is to improve user safety while connecting to public WiFi networks by detecting potential honeypot attacks before a connection is established.
